# Student Result Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a student performance dataset obtained from Kaggle. The goal is to identify factors influencing students' academic performance and discover meaningful insights using statistical analysis and data visualization techniques.

## Dataset

Source: Kaggle Student Performance Dataset

The dataset contains demographic, academic, and socio-economic features affecting student outcomes.

### Features Include

- Gender
- Parental Education
- Ethnic Group
- Weekly Study Hours
- Test Preparation Course
- Marital Status
- Academic Scores

## Objectives

- Understand factors affecting student performance.
- Explore relationships between demographic variables and scores.
- Identify important trends through visualizations.
- Perform statistical analysis on student achievement.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Analysis Performed

- Data Cleaning
- Missing Value Handling
- Descriptive Statistics
- Correlation Analysis
- Distribution Analysis
- Group-wise Performance Comparison
- Data Visualization

## Key Findings

- Students who completed test preparation courses generally performed better.
- Study hours showed a positive relationship with academic scores.
- Parental education demonstrated a noticeable impact on student achievement.
- Several demographic factors exhibited performance variations.
